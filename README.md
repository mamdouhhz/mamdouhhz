## FIELD WORK

#### 🚀 [AWS EKS DevOps Platform](https://github.com/mamdouhhz/nti-final-project)
**Terraform · Jenkins · Ansible · Docker · Kubernetes · AWS**

End-to-end production-style platform: modular Terraform (VPC, auto-scaling EKS, RDS, ECR, Secrets Manager, CloudWatch, Backup) configured via Ansible. Multi-branch Jenkins pipeline with SonarQube gates + Trivy scanning, auto-push to ECR, Helm deploys to EKS. Full observability via Prometheus, Grafana & Loki.

#### 🔐 [Kubernetes Secure Three-Tier App](https://github.com/mamdouhhz/kubernetes-three-tier-app)
**Kubernetes · RBAC · Helm · StatefulSets**

Frontend/backend/MySQL app secured with RBAC, Secrets, and NetworkPolicies. Uses taints, tolerations, node affinity, and PV/PVC storage for scheduling and persistence, with a custom-built backend image and validated external access.

#### 🏗️ [AWS Terraform IaC](https://github.com/mamdouhhz/aws-terraform)
**Terraform · AWS · Multi-AZ**

Modular, highly-available three-tier AWS architecture — custom VPC across multiple AZs, EC2 behind ALBs, and remote state in S3 with locking for safe collaborative infrastructure management.

#### ☁️ OpenShift WordPress Platform
**OpenShift · PVCs · HPA**

Production-style WordPress/MySQL deployment with ConfigMaps, Secrets, health probes, and HPA — exposed via Services/Routes and load-tested.
